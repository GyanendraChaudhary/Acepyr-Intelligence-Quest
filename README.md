# Acepyr Intelligence Quest

Acepyr Intelligence Quest is a Web3-style community engagement dashboard inspired by Acepyr’s intelligence economy.

It is designed to make community participation more interactive, measurable, and rewarding through wallet interaction, daily check-ins, streaks, IQ points, vector missions, reward popups, and leaderboard-style progression.

## Built By

**Anton**

## Features

- Web3 wallet connection
- Daily check-in system
- Streak tracking
- IQ points system
- Acepyr-inspired quest board
- Vector missions inspired by DIONE, THEMIS, KRYPT, JURIS, Consilium, and Foundry
- Reward popup system
- Confetti reward animation
- Community leaderboard concept
- Consilium-style poll demo
- Acepyr branded UI
- Fully responsive colorful interface

## Project Concept

Acepyr is focused on real participation, decision intelligence, user-owned value, and the intelligence economy.

This project brings that idea into a community dashboard where users can participate daily, complete tasks, build streaks, earn points, and explore Acepyr-style decision layers.

The goal is to show how community engagement can become more trackable, interactive, and rewarding.

## Current Version

This is a frontend demo version.

Current data is stored locally in the browser using local storage.
